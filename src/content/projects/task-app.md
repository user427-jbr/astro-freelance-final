## Überblick

TaskFlow ist eine Team-Aufgabenverwaltung, die auf schnelle Kollaboration ausgelegt ist. Änderungen erscheinen in Echtzeit bei allen verbundenen Nutzern, ganz ohne manuelles Aktualisieren.

## Features

- Echtzeit-Synchronisation via WebSocket
- Drag-and-drop Kanban-Board
- Kommentare und Dateianhänge pro Aufgabe
- Rollenbasierte Zugriffssteuerung

## Stack

- **Frontend**: React mit Context API und Framer Motion für Animationen
- **Backend**: Express.js + Socket.io
- **Datenbank**: PostgreSQL mit Prisma ORM
- **Deployment**: Docker + Railway
